# Call Forwarding

- 快速设置手机呼叫转移功能
- 支持不同情景模式
- 支持从通讯录快速选择号码
- 支持历史纪录  
  
脚本界面和功能仿照 [Divert Calls by Mohan Noone](https://itunes.apple.com/cn/app/divert-calls/id625765678?l=en&mt=8), 可点击脚本下方 `Call Forwarding by Neurogram` 下载原版 APP 体验
